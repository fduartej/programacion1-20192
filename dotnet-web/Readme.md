# Web

Principales pasos

1. Crear una carpeta "dotnet-web"

2. Entrar con terminal y ejecutar el siguiente comando

`dotnet new web`

3. Compilar 

`dotnet build`

4. Ejecutar

`dotnet run`

5. Probar en 

`http://localhost:5000`


6. Opcional clean y restore cuando cargamos el proyecto

`dotnet clean`

`dotnet run`
